# Pathum Youth Forum

Official website for **Pathum Youth Forum** — a youth-led platform redefining diplomacy, leadership, and global engagement through Model United Nations conferences and initiatives.

## Live Site

Deployed on Vercel at your custom domain.

## About

Pathum Youth Forum is built on one idea: creating a clear pathway for every young person to grow through diplomacy and dialogue. The flagship conference, **Pathum MUN 2026**, takes place on **23–24 May 2026**.

Founded by **Kasturi Vyawahare**.

## Tech Stack

- Pure HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Outfit + Playfair Display)
- No frameworks, no dependencies — zero build step required
- Vercel-ready static deployment
